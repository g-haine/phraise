## Abstract
This work proposes a time-discretization for nonlinear port-Hamiltonian systems and their controllers using Collocation Methods. First, we provide an overview of port-Hamiltonian systems and present Collocation Methods, detailing their application in deriving discrete-time representations. Next, we propose a time-discretization for controllers using a target system approach. The effectiveness of the proposed methodologies is illustrated through numerical example on a nonlinear port-Hamiltonian system presented by a piezoelectric actuator.
