## Abstract
Model reduction of port-Hamiltonian systems by means the Krylov methods is considered, aiming at structure preservation. It shown how to employ Arnoldi method for model in a particular coordinate system order preserve not only specific number Markov parameters but also reduced model. Furthermore it Lanczos can be applied preserving manner subclass which characterized an algebraic condition. In fact, same and turn out equivalent sense producing models with transfer function.\nKeywords: Port-Hamiltonian systems; reduction; methods; method, method; Structure preservation; parameters.
