## Abstract
This paper deals with the improvement of the induction motor scalar control. All controllers are designed by Passivity Based Control (PBC) and the state variables are flux and speed of the machine. The control law was deducted using the Port Controlled Hamiltonian (PCH) model, and the stability is proven by using Lyapunov theory. The effectivity and robustness of the proposed control scheme are validated by simulations and experimental tests using dSPACE-DSII04 system
